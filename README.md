# 돌봄온

가족들이 사랑하는 노인 분들의 케어 관련 일정을 관리할 수 있도록 설계된 종합적인 웹 애플리케이션입니다. 이 애플리케이션은 진료 예약, 약물 일정, 일상 루틴을 추적하고 케어기버들과 협업할 수 있는 도구를 제공합니다.

## 주요 기능

### 🏠 대시보드
- **오늘의 일정**: 오늘 예정된 모든 예약과 루틴 확인
- **예정된 알림**: 향후 24시간 내 중요한 알림 확인
- **케어 요약**: 총 예약, 활성 약물, 일상 루틴의 빠른 개요
- **케어팀**: 케어에 참여하는 가족 구성원과 케어기버 표시

### 📅 진료 예약
- 상세한 정보와 함께 진료 예약 추가 및 관리
- 의료진/담당자, 장소, 날짜, 시간, 메모 추적
- 맞춤형 알림 설정 (15분~1일 전)
- 상태별 예약 필터링 (예정, 지남, 전체)
- 케어팀원들과 예약 세부사항 공유

### 💊 약물 관리
- 용량과 빈도를 포함한 종합적인 약물 추적
- 여러 일일 복용량 지원 (1회, 2회, 3회, 또는 사용자 지정 시간)
- 약물 코스의 시작일과 종료일 추적
- 활성/비활성 상태 관리
- 약물 알림 및 알림

### ⏰ 일상 루틴
- 일상 케어 루틴 생성 및 관리
- 여러 요일과 함께 유연한 일정 조정
- 루틴 활동을 위한 시간 기반 알림
- 각 루틴 작업에 대한 상세한 설명
- 시간대별 루틴 필터링 (아침, 오후, 저녁)

### 👥 케어팀 관리
- 가족 구성원과 전문 케어기버 추가
- 역할과 권한 할당 (보기, 편집, 알림)
- 연락처 정보 관리
- 케어팀원들과 일정 공유
- 협업 케어 조정

### 🔔 스마트 알림
- 예정된 예약과 약물에 대한 브라우저 알림
- 맞춤형 알림 시간
- 긴급도 기반 우선순위
- 실시간 업데이트 및 알림

## 시작하기

### 사전 요구사항
- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- 추가 소프트웨어 설치 불필요

### 설치
1. 이 저장소를 다운로드하거나 클론
2. 웹 브라우저에서 `home.html` 열기
3. 돌봄온이 모든 기능과 함께 즉시 로드됨

### 첫 설정
1. **예약 추가**: "진료 예약 추가"를 클릭하여 의료 방문 예약
2. **약물 추가**: "약물 추가"를 사용하여 처방전과 용량 추적
3. **루틴 생성**: "루틴 추가"로 일상 케어 루틴 설정
4. **케어기버 초대**: 케어팀에 가족 구성원과 케어기버 추가
5. **알림 활성화**: 알림을 위한 브라우저 알림 허용

## 사용 방법

### 진료 예약 추가
1. 예약 섹션으로 이동
2. "진료 예약 추가" 클릭
3. 필요한 정보 입력:
   - 예약 제목
   - 의료진/담당자 이름
   - 장소
   - 날짜와 시간
   - 메모 (선택사항)
   - 알림 설정
4. "예약 저장" 클릭

### 약물 관리
1. 약물 섹션으로 이동
2. "약물 추가" 클릭
3. 약물 세부사항 입력:
   - 약물명과 용량
   - 빈도 (하루 1회, 2회, 3회, 또는 사용자 지정)
   - 시작일과 종료일
   - 특별 지시사항을 위한 메모
4. 사용자 지정 시간의 경우, 각 복용량에 대한 특정 시간 추가
5. 약물 저장

### 일상 루틴 생성
1. 일상 루틴 섹션 방문
2. "루틴 추가" 클릭
3. 루틴 구성:
   - 루틴 제목과 설명
   - 하루 중 시간
   - 요일
   - 알림 타이밍
4. 루틴 저장

### 케어팀원 추가
1. 케어기버 섹션으로 이동
2. "케어팀원 추가" 클릭
3. 케어기버 정보 입력:
   - 이름, 이메일, 전화번호
   - 역할 (가족 구성원, 전문 케어기버, 간호사, 의사, 기타)
   - 권한 (일정 보기, 일정 편집, 알림 수신)
4. 케어팀에 케어기버 추가

### 일정 공유
1. 모든 항목의 "공유" 버튼 사용
2. 수신자의 이메일 주소 입력
3. 개인 메시지 추가 (선택사항)
4. 공유할 내용 선택 (예약, 약물, 루틴)
5. 초대장 전송

## 데이터 저장

- 모든 데이터는 브라우저의 localStorage에 로컬로 저장
- 외부 서버로 데이터 전송 없음
- 정보는 비공개로 안전하게 유지
- 브라우저 세션 간 데이터 유지

## 브라우저 호환성

- **Chrome**: 완전 지원
- **Firefox**: 완전 지원
- **Safari**: 완전 지원
- **Edge**: 완전 지원
- **모바일 브라우저**: 모바일 기기를 위한 반응형 디자인

## 개인정보 보호 및 보안

- **로컬 저장소**: 모든 데이터는 기기에 로컬로 저장
- **외부 서버 없음**: 외부 서비스로 데이터 전송 없음
- **브라우저 알림**: 선택적 알림 기능
- **가족 제어**: 정보에 대한 접근 권한을 가족이 제어

## 효과적인 사용을 위한 팁

### 가족을 위한
1. **간단하게 시작**: 가장 중요한 예약과 약물부터 시작
2. **모든 사람 참여**: 케어를 돕는 모든 가족 구성원 추가
3. **현실적인 알림 설정**: 일정에 맞는 알림 시간 선택
4. **정기적인 업데이트**: 케어 요구사항이 변경됨에 따라 정보 최신 유지
5. **중요한 정보 백업**: 중요한 일정을 백업으로 인쇄 고려

### 케어기버를 위한
1. **일정 조정**: 공유 기능을 사용하여 모든 사람이 정보를 알 수 있도록 함
2. **변경사항 문서화**: 처방전이 변경될 때 약물 일정 업데이트
3. **비상 연락처**: 모든 케어기버가 중요한 정보에 접근할 수 있도록 보장
4. **정기적인 검토**: 케어 계획의 정기적인 검토 일정

## 문제 해결

### 알림이 작동하지 않음
- 브라우저에서 알림을 허용했는지 확인
- 브라우저가 알림을 지원하는지 확인
- 페이지를 새로고침하고 알림을 다시 허용해보세요

### 데이터가 저장되지 않음
- 브라우저에서 localStorage가 활성화되어 있는지 확인
- 충분한 저장 공간이 있는지 확인
- 브라우저 캐시를 지우고 다시 로드해보세요

### 모바일 표시 문제
- 더 나은 보기를 위해 가로 모드 사용
- 더 나은 가독성을 위해 확대/축소
- 더 쉬운 탐색을 위해 태블릿 사용 고려

## 향후 개선사항

향후 버전을 위한 계획된 기능:
- 더 나은 일정 시각화를 위한 캘린더 뷰
- 의료 기록을 위한 내보내기 기능
- 의료 제공자 시스템과의 통합
- 모바일 앱 버전
- 클라우드 백업 옵션
- 고급 보고서 및 분석

## 지원

질문이나 지원이 필요한 경우:
- 브라우저 콘솔에서 오류 메시지 확인
- 모든 파일이 같은 디렉토리에 있는지 확인
- 다른 브라우저 사용 시도
- 브라우저 캐시 지우기 및 다시 로드
